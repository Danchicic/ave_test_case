# Phone-Address Service API

Микросервис для управления связками **телефон-адрес** с использованием **FastAPI**, **Redis** и **Docker**. Сервис
предоставляет быстрый доступ к данным благодаря кешированию в памяти.

---

## Цель и Назначение

Микросервис разработан для:

- **Кеширования** часто запрашиваемой информации о клиентах
- **Быстрого доступа** к адресам по номерам телефонов
- **Управления** связками телефон-адрес в процессе работы системы
- **Интеграции** в более крупные системы доставки и CRM

---

## Запуск с Docker Compose

```bash
# Запустить сервис
docker-compose up -d
```

**API будет доступен на:** http://localhost:8000 <br>
**Документация (Swagger UI):** http://localhost:8000/docs
---

## Архитектура и Дизайн

### Структура Данных в Redis

```
Key:   phone:<номер_телефона>
Value: <адрес_как_строка>

Пример ключа:
  - phone:+79991234567
```
